Just move the materials to the model and then move the textures to the "albedo" area in the material.
(click on the project version of the model and in the inspector go to "material". You need to put the materials there) 
You must move the texture and the materials with the same name from the field you're moving to.

Apenas mova os materiais para o modelo e então mova as texturas para o campo "albedo" dentro do material.
(clique no modelo na versão original do projeto e no inspector vá na aba "material". Você deve colocar os materiais lá)
Você deve mover as texturas e materiais de mesmo nome que o campo para o qual estiver movendo.
